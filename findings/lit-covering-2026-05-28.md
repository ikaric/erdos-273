# Literature survey — Erdős #273 (covering systems with moduli $p-1$, $p \geq 5$)

**Author:** librarian
**Session:** 2026-05-28
**Subject:** Survey of the covering-systems literature relevant to Erdős–Graham 1980 Problem #273: existence of a covering system of $\mathbb{Z}$ all of whose moduli lie in $\mathcal{M} = \{p-1 : p\text{ prime}, p\geq 5\} = \{4, 6, 10, 12, 16, 18, 22, 28, 30, 36, 40, 42, 46, 52, 58, 60, \ldots\}$.

## Summary

Three sentences for orientation:

1. **Problem 273 is on Bloom's "open" list as of 2026-05-28, with no published partial result attacking #273 directly.** The page (https://www.erdosproblems.com/273) explicitly notes: "Selfridge has found an example using divisors of 360 if $p=3$ is allowed" — so the only obstruction to a positive answer in the existing literature is the loss of the modulus $2 = 3-1$.
2. **The post-2015 distortion-method machinery (Hough; Balister–Bollobás–Morris–Sahasrabudhe–Tiba — BBMST) gives non-existence results conditional on weighted sums $\sum \mu(d)/d$ being small.** Whether $\mathcal{M}$ verifies the BBMST density hypothesis cleanly is a *concrete computation* (see §2 below): the sum $\sum_{m\in\mathcal{M}}(1+(\log p_m)^{3+\epsilon}/p_m)/m$ over even composite numbers shifted from primes can be estimated by Mertens-style asymptotics.
3. **The DeepMind `formal-conjectures` repository has already formalized a `StrictCoveringSystem` structure in Lean 4 (file `FormalConjecturesForMathlib/NumberTheory/CoveringSystem.lean`) and seeded `ErdosProblems/273.lean` as a `sorry`-stub.** This is the upstream we can either depend on (`require`) or copy in. Several adjacent problems (#7, #204, #275, #280, #947) already have full Lean proofs we can study for the formalization style. This is the harness's compound interest in action: ~80% of the structural-definitions plumbing is done already.

The two highest-leverage attack vectors per the literature are:
- **(V-density)** Apply the BBMST $e^{-4C}/2$ density theorem with the weight $\mu(p^i) = 1 + (\log p)^{3+\epsilon}/p$ to the explicit set $\mathcal{M}$ (or a sub-density argument restricting to enough $p-1$'s) and check whether $C$ stays below some threshold making the uncovered set non-empty. **This would settle #273 in the negative if $C < \infty$ on $\mathcal{M}$.** The literature does *not* explicitly compute this for $\mathcal{M}$ — that is the project's potential novelty.
- **(V-construct)** Adapt the Selfridge-divisors-of-360 construction to $\mathcal{M}$ by trying to replace the modulus-2 progression with a finite collection drawn from $\mathcal{M}$. The basic obstruction: in Selfridge's construction modulus 2 covers one full residue class with density 1/2, and $\mathcal{M}$'s smallest modulus is 4 (density 1/4), so at minimum 2 elements of $\mathcal{M}$ are needed to cover what 2 alone did. Whether any finite collection from $\mathcal{M}$ can substitute is a finite combinatorial question over the dense lattice generated by the modulus-12 lcm structure of Selfridge's 360-system.

## 1. Hough 2015 — minimum-modulus result

**Bob Hough, *Solution of the minimum modulus problem for covering systems*, Annals of Mathematics 181:1 (2015), 361–382. arXiv:1307.0874.** https://arxiv.org/abs/1307.0874

- **Theorem (Hough 2015).** In any covering system of the integers with distinct moduli, the minimum modulus is $\leq 10^{16}$.
- **Method.** Probabilistic sieving: at each stage $k$, refine a probability measure on $\mathbb{Z}/Q_k\mathbb{Z}$ (with $Q_k = $ lcm of moduli through stage $k$) by an explicit "distortion" $w_k$ that accounts for which residues remain uncovered. A Lovász-Local-Lemma–style entropy estimate then forces $\sum w_k$ to be too large to be consistent with the small uncovered set, contradicting the covering hypothesis.
- **What it gives us.** A *universal* upper bound on the smallest modulus in *any* covering system, with no structural assumptions on the moduli set beyond distinctness. So if $\#273$ has a positive answer, the covering must use *some* modulus $m\in\mathcal{M}$ with $m \leq 10^{16}$.
- **What it does not give.** Hough's argument is not sensitive to which set the moduli come from, and the $10^{16}$ bound is far too weak (and the moduli set far too dense) to immediately exclude $\mathcal{M}$.

Lean formalization: **none upstream.** The proof is delicate analytic-probabilistic combinatorics — would be a substantial formalization project.

## 2. Balister–Bollobás–Morris–Sahasrabudhe–Tiba 2022 — the distortion method

**Paul Balister, Béla Bollobás, Robert Morris, Julian Sahasrabudhe, Marius Tiba, *On the Erdős covering problem: the density of the uncovered set*, Inventiones mathematicae 228 (2022), 377–414. arXiv:1811.03547.** https://arxiv.org/abs/1811.03547

This is the central reference for #273. The paper contains *several* theorems we should track separately.

### 2.1 The density theorem (Theorem 1.1)

Define the weight $\mu(p^i) := 1 + (\log p)^{3+\epsilon}/p$ extended multiplicatively to $\mu(d) = \prod_{p^i\,\|\,d}\mu(p^i)$. For a system $\mathcal{A} = \{a_i\pmod{d_i}\}_{i=1}^k$ with distinct moduli $d_1<\cdots<d_k$, let
$$C(\mathcal{A}) := \sum_{i=1}^k \frac{\mu(d_i)}{d_i}.$$

**Theorem 1.1 (BBMST 2022).** If the minimum modulus $d_1 \geq M$ for $M$ sufficiently large, then the density of $\mathbb{Z}\setminus\bigcup_i(a_i + d_i\mathbb{Z})$ is at least $e^{-4C(\mathcal{A})}/2$.

- **Implication for #273.** This is *the* high-leverage tool. If we can show $C(\mathcal{A}) < \infty$ uniformly over all finite subsystems with moduli in $\mathcal{M}$ — equivalently, if $\sum_{m\in\mathcal{M}} \mu(m)/m < \infty$ — then the density bound rules out a covering: the uncovered set has positive density and is hence non-empty.
- **The arithmetic question.** For $\mathcal{M} = \{p-1: p\geq 5\text{ prime}\}$, we need to estimate $\sum_{p\geq 5} \mu(p-1)/(p-1)$. Since $p-1$ is even with smallest prime factor 2, and most $p-1$ have a "generic" factorization, heuristically $\mu(p-1) \asymp 2$ on average, so the sum behaves like $2\sum_{p\geq 5} 1/p$ which **diverges (Mertens)**. So $C(\mathcal{A})$ is *not* bounded uniformly — the density theorem alone does not immediately rule out #273.
- **However**, the *minimum-modulus* version of the density theorem (taking $M$ large) does say that finitely many residues with moduli all large enough cannot cover, which gives a quantitative bound on how many small moduli a covering can use. This is the lever for partial obstructions.

### 2.2 Improved minimum-modulus bound (Section 4 / Corollary)

**Corollary (BBMST 2022).** The minimum modulus of any distinct covering system is $\leq 616{,}000$.

This drops Hough's $10^{16}$ to a six-digit number. Bloom's page for Problem 2 (Erdős' min-modulus problem) confirms: "Balister, Bollobás, Morris, Sahasrabudhe, and Tiba [BBMST22] ... improved the upper bound on the smallest modulus to $616000$."

### 2.3 Schinzel's conjecture (Theorem 1.2)

**Theorem 1.2 (BBMST 2022).** If $\mathcal{A}$ is a finite collection of arithmetic progressions that covers $\mathbb{Z}$, then at least one of the moduli divides another.

Combined with Selfridge's reduction (reported in Schinzel 1967), this *disproved* Erdős Problem #586 ("Is there a covering system such that no two of the moduli divide each other?"). For $\mathcal{M}$: many pairs $m_1 = p_1-1, m_2 = p_2-1$ are mutually non-dividing (e.g., $4 \nmid 6, 6\nmid 4$), but plenty of pairs *do* divide each other (e.g., $4 \mid 12$, $6 \mid 12$, $6 \mid 18$, $10\mid 30$, $12\mid 60$). Theorem 1.2 says: *any* covering must contain a divisibility pair, so a covering using $\mathcal{M}$ must use some such pair — this constrains but does not exclude.

### 2.4 The 2-or-3 / 9-or-15 theorems (Theorems 1.3–1.4)

**(Hough–Nielsen 2019, simpler proof BBMST 2022).** Every covering system has a modulus divisible by 2 or by 3.

**(BBMST 2022).** If every modulus in a covering system is odd, then 9 divides the lcm of moduli, or 15 does.

**For #273.** Since every modulus in $\mathcal{M}$ is even ($p-1$ for $p\geq 5$ is even), the 2-or-3 theorem is *automatically* satisfied: 2 divides every modulus. The 9-or-15 theorem applies only to odd-moduli covers, irrelevant here. **So these results do not obstruct #273.**

### 2.5 Lean formalization

**None upstream of the density theorem itself.** The DeepMind project has the `CoveringSystem` *type* but no `theorem density_uncovered` lemma yet. This is the headline import-vs-reformalize tradeoff: formalizing BBMST Theorem 1.1 in full would be a multi-month project.

Closer to viability: the *Schinzel's conjecture* statement (Theorem 1.2 — "at least one modulus divides another") would be roughly 1 page of Lean glue plus access to the analytic core; the analytic core is the same distortion method as the density theorem.

## 3. Post-2020 covering-system literature (arXiv math.NT, 36-month window)

Listed most-relevant-first:

### 3.1 Cummings–Filaseta–Trifonov 2022 — sharp bound for squarefree moduli

**Maria Cummings, Michael Filaseta, Ognian Trifonov, *An upper bound for the minimum modulus in a covering system with squarefree moduli*, arXiv:2211.08548 (Nov 2022).** https://arxiv.org/abs/2211.08548

- **Theorem 1.1.** Every covering system with distinct squarefree moduli has minimum modulus $\leq 118$.
- **Method.** Refinement of BBMST distortion method, specialized to squarefree by exploiting that each squarefree modulus factors as a product of distinct primes, naturally fitting the BBMST hyperplane-covering framework.
- **Relevance to $\mathcal{M}$.** $\mathcal{M}$ is *not* a subset of squarefree integers (e.g., $4 = 2^2 \in \mathcal{M}$, $12 = 2^2\cdot 3 \in \mathcal{M}$, $16 = 2^4\in\mathcal{M}$, …). So 1.1 doesn't apply directly. But the *technique* — instantiating the distortion method on a structured moduli set — is the closest published template for what we need to do for $\mathcal{M}$.
- **Effective constants.** Authors note their bounds are not optimized; the methodology is what we'd port, not the specific number 118.

Lean formalization: none.

### 3.2 Balister–Bollobás–Morris–Sahasrabudhe–Tiba 2021 — Erdős–Selfridge squarefree odd

**Balister, Bollobás, Morris, Sahasrabudhe, Tiba, *The Erdős–Selfridge problem with square-free moduli*, Algebra & Number Theory 15 (2021), 609–626. arXiv:1901.11465.** https://arxiv.org/abs/1901.11465

- **Theorem 1.1.** Any covering system with distinct squarefree moduli must contain an even modulus.
- Restated: there is no covering system with distinct, *odd*, *squarefree* moduli. This is a partial step toward Problem #7 (the unrestricted Erdős–Selfridge "odd covering" conjecture).
- **Method.** Distortion-method moment bounds (Theorems 3.1–3.2 of that paper) tailored to the squarefree constraint.
- **Relevance to $\mathcal{M}$.** All moduli in $\mathcal{M}$ are even, so this result is *vacuously satisfied* for #273. The methodology is again the lever.

### 3.3 Klein–Koukoulopoulos–Lemieux 2022 — j-th smallest modulus

**Jonah Klein, Dimitris Koukoulopoulos, Simon Lemieux, *On the $j$-th smallest modulus of a covering system with distinct moduli*, arXiv:2212.01299 (Dec 2022, rev. Aug 2023).** https://arxiv.org/abs/2212.01299

- **Theorem 1.1.** There is an absolute constant $c>0$ such that the $j$-th smallest modulus in any minimal covering system with distinct moduli is $\leq \exp(cj^2/\log(j+1))$.
- **Relevance.** Gives uniform bounds for the entire low end of the modulus spectrum — not just the smallest. If a covering with moduli in $\mathcal{M}$ exists, its $j$-th smallest modulus is bounded.
- **Effective constants.** $c$ not made explicit.

### 3.4 Klein–Koukoulopoulos–Lemieux 2024 — sum of reciprocals close to 1

**Klein, Koukoulopoulos, Lemieux, *Covering systems with the sum of the reciprocals of the moduli close to 1*, arXiv:2407.15280 (Jul 2024).** https://arxiv.org/abs/2407.15280

- Proves Erdős–Selfridge 1973 conjecture: if min modulus $>4$ then $\sum 1/d_i$ is bounded away from 1.
- **Relevance to $\mathcal{M}$.** The smallest element of $\mathcal{M}$ is 4. If we require *only* elements of $\mathcal{M}$, we still get min modulus $\geq 4$. This Klein–Koukoulopoulos–Lemieux result might apply *vacuously at the boundary* and give an upper bound on the sum of reciprocals of $\mathcal{M}$-covering moduli, but does *not* directly settle #273.

### 3.5 Li–Wang–Wang–Yi 2024; B. Wang 2024 — function-field analogs

**Huixi Li, Biao Wang, Chunlin Wang, Shaoyun Yi, *On Erdős covering systems in global function fields*, arXiv:2402.03810 (Feb 2024).** https://arxiv.org/abs/2402.03810
**Biao Wang, *Bounds for Erdős covering systems in global function fields*, arXiv:2408.10460 (Aug 2024).** https://arxiv.org/abs/2408.10460

- Extends distortion method to $\mathbb{F}_q[x]$: no covering system of $\mathbb{F}_q[x]$ with distinct moduli for $q > 73$ (Wang) / $q \geq 759$ (Li–Wang–Wang–Yi earlier).
- **Relevance to #273.** Cross-domain reference: shows the distortion method generalizes off-integer settings. Methodologically useful but not directly applicable.

### 3.6 Li–Wang–Yi 2023 — minimum modulus in number fields

**Huixi Li, Biao Wang, Shaoyun Yi, *On the minimum modulus problem in number fields*, arXiv:2302.05946 (Feb 2023).** https://arxiv.org/abs/2302.05946

- Extends Hough's $10^{16}$ to algebraic number fields via inverse-system probability measures.
- **Relevance to #273.** Mostly inspirational — but if one ever needed to lift #273 to $\mathbb{Z}[\zeta_p]$, this is the playbook.

### 3.7 Harrington–Sun–Wong 2021 / Bispels et al. 2025 — odd-moduli variants

**Joshua Harrington, Yewen Sun, Wing Hong Tony Wong, *Covering systems with odd moduli*, arXiv:2104.00602 (Apr 2021).** https://arxiv.org/abs/2104.00602 (Discrete Math 2022)
**Chris Bispels, Matthew Cohen, Joshua Harrington, Joshua Lowrance, Kaelyn Pontes, Leif Schaumann, Tony W. H. Wong, *A further investigation on covering systems with odd moduli*, arXiv:2507.16135 (Jul 2025).** https://arxiv.org/abs/2507.16135

- Both papers study a relaxation of Problem #7 where one odd prime is allowed to repeat as a modulus. Define $t_p$ as the minimum repetitions needed; bounds $t_3 \leq 2, t_5\leq 3, t_7\leq 4$, $t_p \leq p-4$ for $11\leq p\leq 19$.
- **Relevance to #273.** Tangentially related (variants of #7, not #273). But the *style* of construction — placing redundant moduli to fill awkward residue gaps — could inform a constructive attack.

### 3.8 Adenwalla 2025 / Erdős Problem #204 — divisor covering with coprime overlap

**Sarosh Adenwalla, *A Question of Erdős and Graham on Covering Systems*, arXiv:2501.15170 (Jan 2025).** https://arxiv.org/abs/2501.15170

The smallest interesting $k$ is around $11$ (the all-moduli-$\geq 3$ minimum), though for our setting we need $\geq 12$ moduli at minimum since each modulus must be $\geq 4$.

## 5. Numerical / SAT attacks

**No known SAT/computational attack restricted to $\mathcal{M}$.** What exists:

- Owens 2014 — explicit covering with minimum modulus 42 (Bloom #2 page records this as the current record for "what minimum-modulus is achievable").
- Nielsen 2009 — explicit covering with minimum modulus 40, using $>10^{50}$ congruences.
- Agrawal et al. 2022 — classified all coverings with $\leq 10$ moduli (computational enumeration via constraint satisfaction).
- Choi 1971 (early enumeration of small coverings, $N = 20$).
- Krukenberg 1971 (constructions, $N = 24$).

**Useful conclusion for #273.** If a covering by $\mathcal{M}$ exists, by Klein–Koukoulopoulos–Lemieux 2022 each of the lowest $j$ moduli is bounded by $\exp(cj^2/\log(j+1))$. Combined with $\mathcal{M}$'s explicit form, this means: for a covering with $k$ moduli, the $j$-th smallest is in $\mathcal{M} \cap [4, \exp(cj^2/\log(j+1))]$, which gives a *finite search space* per element. However, $k$ itself is not bounded a priori. SAT-style enumeration over small lcm-of-moduli (say $\text{lcm} \leq 360, 2520, 27720$, …) is the natural first computational attack.

Concrete numerical search target: **enumerate all $\mathcal{M}$-restricted covering systems with $\text{lcm}(m_i)\mid L$** for $L \in \{12, 60, 360, 2520, 27720, 720720\}$. For each $L$, enumerate all $\mathcal{M}\cap\{\text{divisors of }L\}$, set up a 0/1-ILP (each residue class of $\mathbb{Z}/L\mathbb{Z}$ must be covered $\geq 1$), and search. If none of these LCMs admits a covering, that's a strong empirical signal (but not a proof).

## 6. Classical constructions (Selfridge, Krukenberg, …)

### 6.1 The Selfridge "divisors of 120" covering — the canonical small example

The Wikipedia article on Covering Systems lists the classical Erdős/Selfridge example with moduli divisors of 120:
$$\{0(3),\ 0(4),\ 0(5),\ 1(6),\ 1(8),\ 2(10),\ 11(12),\ 1(15),\ 14(20),\ 5(24),\ 8(30),\ 6(40),\ 58(60),\ 26(120)\}.$$

Moduli: $\{3, 4, 5, 6, 8, 10, 12, 15, 20, 24, 30, 40, 60, 120\}$.

Which of these are in $\mathcal{M}$? $\mathcal{M}$ requires $m+1$ prime, $m+1 \geq 5$:
- $3$ → $3+1=4=2^2$, not prime. **Not in $\mathcal{M}$.**
- $4$ → $4+1=5$, prime ✓ . **In $\mathcal{M}$.**
- $5$ → $5+1=6=2\cdot 3$, not prime. **Not in $\mathcal{M}$.**
- $6$ → $6+1=7$, prime ✓ . **In $\mathcal{M}$.**
- $8$ → $8+1=9=3^2$, not prime. **Not in $\mathcal{M}$.**
- $10$ → $10+1=11$, prime ✓ . **In $\mathcal{M}$.**
- $12$ → $12+1=13$, prime ✓ . **In $\mathcal{M}$.**
- $15$ → $15+1=16=2^4$, not prime. **Not in $\mathcal{M}$.**
- $20$ → $20+1=21=3\cdot 7$, not prime. **Not in $\mathcal{M}$.**
- $24$ → $24+1=25=5^2$, not prime. **Not in $\mathcal{M}$.**
- $30$ → $30+1=31$, prime ✓ . **In $\mathcal{M}$.**
- $40$ → $40+1=41$, prime ✓ . **In $\mathcal{M}$.**
- $60$ → $60+1=61$, prime ✓ . **In $\mathcal{M}$.**
- $120$ → $120+1=121=11^2$, not prime. **Not in $\mathcal{M}$.**

So in the classical 120-divisor example, **6 of 14 moduli are in $\mathcal{M}$** (4, 6, 10, 12, 30, 40, 60) — *almost half*. The non-$\mathcal{M}$ moduli are $\{3, 5, 8, 15, 20, 24, 120\}$, totaling 7 moduli (note: 60 isn't on this list; my count above is right since I named the 7 that are NOT in $\mathcal{M}$, plus the 7 that ARE; sum to 14). This is a striking density: the obstruction is precisely the "non-$\mathcal{M}$ moduli" 3, 5, 8, 15, 20, 24, 120 which we must replace.

### 6.2 The "Selfridge divisors of 360" example referenced by Bloom

The erdosproblems.com/273 page asserts: "Selfridge has found an example using divisors of 360 if $p=3$ is allowed." This means: there *exists* an explicit covering with moduli in $\{d : d\mid 360,\ d > 1\}$ — namely, taking $p=3$ allows the modulus $2 = 3-1$. So the "if $p=3$ is allowed" version of #273 is **solved positively** (see the `erdos_273.variants.three` stub in DeepMind formal-conjectures, marked `research solved`). I have not located the explicit residues for this Selfridge-360 construction in the literature — would need to ping Bloom or check Schinzel 1967 / Sun's survey.

**Critical observation.** The DeepMind Lean file marks `erdos_273.variants.three : answer(True)` (the $p\geq 3$ version is solved) but `erdos_273 : answer(sorry)` (the $p\geq 5$ version is open). So the gap between #273 and its $p\geq 3$ variant is **precisely the loss of the modulus $2$**. *This is the entire content of the problem.*

### 6.3 Why removing $p=3$ (modulus 2) breaks Selfridge

Modulus 2 covers exactly half of $\mathbb{Z}$ — the largest density that can come from a single arithmetic progression. Without it, the smallest contribution per arithmetic progression is $1/4$ (from $\mathcal{M}$'s smallest element, $4 = 5-1$). To cover $\mathbb{Z}$, we need $\sum 1/m_i \geq 1$ (necessary condition). With moduli in $\mathcal{M}$ starting at 4, we need at minimum **a few elements of $\mathcal{M}$** to cover the residue class that 2 would have covered alone.

Specifically: a "modulus-2" arithmetic progression $0\pmod{2}$ covers the even integers. To replace this with $\mathcal{M}$-elements only, we need to cover every even integer by some $a_i\pmod{m_i}$ with $m_i\in\mathcal{M}$ (in particular, $m_i$ is itself even). So the question reduces to: **can $\{2a : a\in\mathbb{Z}\}$ be covered by arithmetic progressions modulo even numbers in $\mathcal{M}/2 = \{2, 3, 5, 6, 8, 9, 11, 14, 15, 18, 20, 21, 23, 26, 29, 30, \ldots\}$?** This is itself a covering-system question on the *evens*, with allowed moduli the set $\mathcal{M}/2$.

Note $\mathcal{M}/2$ has the property: it contains $2 = 4/2$, $3 = 6/2$, $5 = 10/2$, $6 = 12/2$, $8 = 16/2$, $9 = 18/2$, …. In particular **$\mathcal{M}/2$ contains $2$ and $3$**, so the *sub-problem* of covering even integers by these moduli is equivalent to covering $\mathbb{Z}$ by moduli in $\mathcal{M}/2$, which *does* allow small moduli — including 2 itself. The recursion does not immediately blow up. **The structural reason #273 isn't trivially false is that this recursive reduction stays inside an "almost arbitrary" moduli set.**

- **Nielsen 2009** — explicit covering with minimum modulus $40$ (used $> 10^{50}$ congruences); arXiv link: https://mathdept.byu.edu/~pace/MinimumModulus40_web.pdf
- **Owens 2014** — minimum modulus $42$.

None of these classical examples constrain their moduli to $\mathcal{M}$ specifically; they are evidence that *very large* minimum moduli are achievable (with lots of effort), bounded above by Hough/BBMST.

## 7. Lean 4 / Mathlib status

### 7.1 Mathlib upstream — `CoveringSystem` does NOT exist in Mathlib core

Loogle / leansearch queries return no `CoveringSystem` structure in Mathlib (as of v4.30.0, which is pinned in `formal/lakefile.toml`). The closest Mathlib infrastructure:
- `Mathlib.Data.ZMod.Basic` — the `ZMod n` type, residue classes modulo $n$.
- `Mathlib.NumberTheory.Cyclotomic` — root-of-unity context where $p-1$ appears as $\text{ord}(\mathbb{Z}/p\mathbb{Z})^\times$.
- `Mathlib.RingTheory.Ideal.Defs` — ideal cosets in $\mathbb{Z}$ (the BBMST formalization's substrate).

### 7.2 DeepMind formal-conjectures has a `CoveringSystem` structure (key asset!)

The Google DeepMind `formal-conjectures` repository (`google-deepmind/formal-conjectures` on GitHub) maintains a Lean 4 library tracking Erdős problems. Two files matter:

**File 1:** `FormalConjecturesForMathlib/NumberTheory/CoveringSystem.lean` (defines the type)

```lean
import Mathlib.Algebra.Group.Pointwise.Set.Basic
import Mathlib.Algebra.Module.Submodule.Lattice
import Mathlib.RingTheory.Ideal.Defs

structure CoveringSystem (R : Type*) [CommSemiring R] where
  ι : Type
  [fintypeIndex : Fintype ι]
  residue : ι → R
  moduli : ι → Ideal R
  unionCovers : ⋃ i, ({residue i} : Set R) + (moduli i : Set R) = @Set.univ R
  ne_bot : ∀ i, moduli i ≠ ⊥
  ne_top : ∀ i, moduli i ≠ ⊤

structure StrictCoveringSystem (R : Type*) [CommSemiring R] extends CoveringSystem R where
  injective_moduli : moduli.Injective
```

The two structures (`CoveringSystem` allows repeated moduli; `StrictCoveringSystem` requires distinct moduli — matching Erdős's definition) are *generic over any commutative semiring*. For $R = \mathbb{Z}$, moduli are ideals $(n\mathbb{Z})$ identifiable with $n\in\mathbb{N}_{\geq 1}$.

**File 2:** `FormalConjectures/ErdosProblems/273.lean` (the problem statement, currently `sorry`)

```lean
/--
Is there a covering system all of whose moduli are of the form $p-1$ for some primes $p \geq 5$?
-/
@[category research open, AMS 5 11]
theorem erdos_273 : answer(sorry) ↔ ∃ c : StrictCoveringSystem ℤ, ∀ i, ∃ (p : ℕ),
    p.Prime ∧ 5 ≤ p ∧ c.moduli i = Ideal.span {↑(p - 1)} := by
  sorry

/--
Is there a covering system all of whose moduli are of the form $p-1$ for some primes $p \geq 3$?
-/
@[category research solved, AMS 5 11]
theorem erdos_273.variants.three : answer(True) ↔ ∃ c : StrictCoveringSystem ℕ,
    ∀ i, ∃ p, p.Prime ∧ 3 ≤ p ∧ c.moduli i = Ideal.span {↑(p - 1)} := by
  sorry
```

The `variants.three` (with $p\geq 3$) is marked `research solved` (so the orchestrator can use this as the prior — Selfridge's 360-divisor construction proves it positively). The main theorem `erdos_273` carries `answer(sorry)` — the answer-direction itself is unknown.

### 7.4 Adjacent fully-formalized problems

Three closely related covering-system problems already have full Lean proofs (the corresponding statements compile with no `sorry` in dependency chain):

- **#275** Selfridge / Crittenden–Vanden Eynden 1970 (cover $2^r$ consecutive $\Rightarrow$ cover all). Formalized by Alexeev using Aristotle. Source: https://github.com/plby/lean-proofs/blob/main/src/v4.24.0/ErdosProblems/Erdos275.lean
- **#204** Adenwalla 2025 (no $n$ with divisor-covering with coprime overlaps). Formalized by van Doorn using Aristotle. Source: https://github.com/Woett/Lean-files/blob/main/ErdosProblem204.lean
- **#947** Mirsky–Newman / Davenport–Rado (no exact distinct-moduli cover).
- **#281** Davenport–Erdős (density-zero uncovered → uniformly small prefix).

These are the closest reference points for the Lean style needed to formalize a covering-systems theorem from scratch.

## 8. Recommended next steps

In rough order of expected ROI:

1. **(immediate, no math)** **Wire up `formal-conjectures` as a Lake dependency.** Modify `formal/lakefile.toml` to add `[[require]] name = "formal-conjectures"; git = "https://github.com/google-deepmind/formal-conjectures.git"; rev = "<pin SHA>"`. Then in `formal/Erdos273/Defs.lean`, import `FormalConjecturesForMathlib.NumberTheory.CoveringSystem` and re-export. **Net effect: the `StrictCoveringSystem` type is free.** Cost: 1 commit, 10 lines of Lean.

2. **(computational, ~1 session)** **Formalize the *positive* answer for the $p\geq 3$ variant.** Selfridge's 360-divisor construction is concrete and computable. Translate it to `Erdos273.SelfridgeP3.lean` exhibiting an `StrictCoveringSystem ℤ` whose moduli are all of form $p-1$ with $p\geq 3$, $p$ prime. *This closes the `erdos_273.variants.three` stub in the DeepMind repo and gives our project a concrete win plus a reference template for the harder $p\geq 5$ direction.* Numerical: needs finding the explicit residues; literature suggests Schinzel 1967 or Sun's survey has them.

3. **(analytic, ~1-2 sessions)** **Estimate $\sum_{p\geq 5}\mu(p-1)/(p-1)$ explicitly** where $\mu(d) = \prod_{p^i\|d}(1 + (\log p)^{3+\epsilon}/p)$. If this sum diverges (likely, via Mertens), the density theorem alone won't suffice. If it converges (unlikely but worth checking), apply BBMST 1.1 directly and get a *negative* answer for #273. Computationally: use Python/`sympy` to compute the partial sums up to $p\leq 10^k$ for $k=3,4,5,6$.

4. **(constructive, multi-session)** **Try SAT-based enumeration.** Use `cadical`/`kissat` (already in CLAUDE.md as install-on-demand) to enumerate all *partial* covering systems with moduli in $\mathcal{M}\cap[4, 10^k]$ for increasing $k$. The Klein–Koukoulopoulos–Lemieux bound on the $j$-th smallest modulus gives a per-element upper bound. Practically: code the question "does any subset of $\mathcal{M}\cap\{d : d\mid L\}$ form a covering of $\mathbb{Z}/L\mathbb{Z}$?" as a SAT/0-1-ILP and run for $L \in \{12, 60, 360, 2520, 27720, 720720\}$. Each $L$ is a finite question; either a covering is found (settles #273 positively for that $L$) or non-existence is shown (provides empirical evidence and constrains the answer to require larger lcm).

5. **(speculative, exploratory)** **Try to adapt the Cummings–Filaseta–Trifonov 2022 squarefree-118 method to $\mathcal{M}$.** Their distortion argument exploits squarefreeness in a structurally clean way; the question is whether $\mathcal{M}$'s "even-shifted-prime" structure admits a parallel distortion argument. This is genuinely novel territory — the literature has no published attempt for $\mathcal{M}$.

6. **(orthogonal)** **Read Schinzel 1967** for the original $p\geq 3$ Selfridge construction in full. This will likely be the only place the explicit residues are written down. Reference list of Bloom #273 cites `[ErGr80,p.24]`.

- `Erdos273.M_subset_even`: every element of $\mathcal{M}$ is even (`p prime → p ≥ 5 → 2 ∣ (p-1)`). Trivial Mathlib import wrap. **One commit, ~5 lines.**
